# CannyEdge

## Description

Canny Edge detection image filter with Cuda at scale (batch files processing).

## Prerequisites
```
    Ubuntu 22.04
    CUDA 12.3
    CUDNN 8.9.5.29
```

## Build and Run
```
    $ ./run.sh
```
